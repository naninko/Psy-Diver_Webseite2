# PSY-DIVER Website

Official website for the PSY-DIVER research project - Gender and Diversity-Related Factors in Psychiatric-Psychotherapeutic Hospital Care.

## Project Overview

PSY-DIVER is a 3-year research initiative (November 2025 - October 2028) examining how gender, origin, social circumstances, and living situations influence psychiatric treatment patterns, effectiveness, and outcomes.

**Lead Institution:** LVR-Institut für Forschung und Bildung, LVR-Klinik Köln
**Funding:** €1 Million from Innovationsfonds (ID: 01VSF25022)
**Principal Investigator:** Prof. Dr. Euphrosyne Gouzoulis-Mayfrank

## Website Features

- Modern React Application built with Vite
- **Bilingual Support:** German (DE) and English (EN) with language switcher
- Responsive Design optimized for mobile, tablet, and desktop
- Accessibility Compliant (WCAG 2.1 AA standards)
- LVR Corporate Design color palette and typography
- Multi-page Structure: Home, About, Partners, News, Contact

## Technology Stack

- **Framework:** React 18
- **Build Tool:** Vite
- **Routing:** React Router DOM
- **Internationalization:** i18next & react-i18next
- **Styling:** CSS with custom properties
- **Typography:** Source Sans 3 (Google Fonts)

## Installation & Setup

### Install Dependencies
```bash
npm install
```

### Development Server
```bash
npm run dev
```
The site will be available at `http://localhost:5173/`

### Production Build
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```
